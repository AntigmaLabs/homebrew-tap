# homebrew-tap
binary distribution

> ⚠️ **NOTE**: The brew tap step is required only once.

## Using Homebrew
- make sure you have [Homebrew](https://brew.sh/) installed
- add the tap
```sh
brew tap AntigmaLabs/homebrew-tap
```

## Install
```sh
brew install ante
```

## Uninstall
```sh
brew uninstall ante
```
## Run
```sh
ante
```
